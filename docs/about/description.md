# Sobre o Trabalho

Ao longo da disciplina, iremos desenvolver de forma incremental um **Sistema de Gestão de Eventos**, aplicando conceitos fundamentais de construção de software, arquitetura de aplicações e reutilização de componentes.

O objetivo do trabalho é simular o desenvolvimento de um sistema real, dividido em múltiplos componentes que se comunicam entre si, seguindo princípios de **modularidade, reutilização e separação de responsabilidades**.

O sistema permitirá gerenciar eventos, participantes e inscrições, sendo composto por três partes principais:

- **Serviço de Autenticação e Autorização**
- **Microsserviços**
- **Cliente**

Cada parte será desenvolvida em uma etapa diferente do semestre, representando as avaliações **T1, T2 e T3**.

A proposta segue uma abordagem incremental: cada trabalho **reutiliza e expande o que foi construído anteriormente**, permitindo que pratiquemos a evolução de sistemas ao longo do tempo, uma habilidade essencial na construção de software profissional.
